☁️ Google Sheets Cloud Sync
Connect to Google Sheets for free cloud backup. Your data syncs automatically and is accessible from any device. See the setup guide below.
📋 Setup Guide (5 min, one-time)
1. Open Google Sheets → New blank spreadsheet
2. Go to Extensions → Apps Script
3. Paste the backend code (shared separately)
4. Click Run → setupSheets (creates tabs)
5. Click Deploy → New deployment → Web app
6. Set "Who has access" to Anyone
7. Copy the URL and paste above ↑
8. Click "Test Connection" — done! 🎉
